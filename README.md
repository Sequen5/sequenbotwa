```bash
> git clone https://github.com/Sequen5/sequenbotwa
> cd whatsapp-bot
```

## âœï¸ Edit filenya
Edit value yang diperlukan di `settings/setting.json`.
```json
{
    "limitCount": 10,
    "ownerNumber": "",
    "memberLimit": 10,
    "groupLimit": 50,
    "medialimitCount": "5",
    "prefix": "/",
    "vhtearkey": "Your Apikey In Here",
    "itech": "Your Apikey In Here"
}

```

`ownerBot`: your WhatsApp number.  
`prefix`: bot's prefix.  
`itech`: API token. Anda bisa mendapatkannya [di sini] (https://api.i-tech.id) dengan membuat akun. Setelah itu, setel IP statis server / host Anda

`vhtearkey`: VHTear API token. Anda bisa mendapatkannya [di sini] (https://api.vhtear.com/) dengan membeli kunci API miliknya.

## ğŸ” Menginstal dependensi
```bash
> npm install
> npm start
```
